# Historical archive-packaging note

This note records the repository's early archive-packaging plan before the final evidence model and publication policy were established.

The early plan was to preserve report timestamps, create a `.tar.gz` of the project directory, and attach it to a GitHub Release.

That early snapshot is now classified as a historical Part I-era snapshot, not as an authoritative whole-project archive. It predates Part II integration, the Part III execution pilots, the runtime-reset finding, the Foundry rerun, and the Session Workflow Playbook.

The final project model separates Git history, off-runtime raw evidence backup, and curated evidence publication. The historical filename remains unchanged for provenance:

`chatgpt-session-repo-workflow-study-20260707.tar.gz`

Do not describe that file as a current repository archive. Current source snapshots are provided by tagged repository states, while large pilot evidence bundles are tracked under `release-assets/`.
