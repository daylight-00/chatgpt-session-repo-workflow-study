# Legacy and historical snapshots

This directory documents superseded packaging states. Large historical binaries are not committed here.

The file named `chatgpt-session-repo-workflow-study-20260707.tar.gz` should be treated as an **early Part I-era snapshot**, not as the authoritative current project archive. It predates later meta-observability documentation and the Part III execution pilots.

Git history and the current structured documentation are authoritative for the evolving study narrative. Large complete pilot bundles, when available and redistribution-reviewed, belong in GitHub Releases and are tracked in `release-assets/MANIFEST.tsv`.
